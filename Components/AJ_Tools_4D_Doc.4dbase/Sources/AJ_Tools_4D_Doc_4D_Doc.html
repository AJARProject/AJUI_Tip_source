AJ_Tools_4D_Doc_generate_doc

This method will generate a Doc in HTML based on the method headers

#NameTypeOptionalDescription
$1$paramsobjectTrueobject containing the path to store the doc and if we want to export in dark mode

The generated 4D_Doc will be generated at the given path or in the 4D Database in a "Sources/4D_Doc/" folder.
You can choose to export the 4D_Doc in dark mode
$params can contain :
{
  "path_storeDoc":<path>,
  "darkMode":<True/False>,
}

AJ_Tools_4DDoc_extractAllHeader

This method will return a description of the methods based on the header generated by the Macro "_head"

#NameTypeOptionalDescription
$0$methodsobject - all methods description

The $methods object will contain, for each method, a description of the method.
The description is based on a header formatted with the "_head" macro given with the 4D_Doc component
The macro should not be changed because the doc will be generated based on the macro structure
